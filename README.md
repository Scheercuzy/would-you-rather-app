# Would You Rather

Would You Rather is a simple React Js application in which you answer a question from 2 options.

- Create your own question
- A leaderBoard with all the users stats

## Installing

```bash
npm install
```

## Running the application

```bash
npm start
```

## Usage

Once the above steps are done, a browser window should open up with the app. Incase a new browser window doesn't show, head to [localhost:3000](http://localhost:3000) to interact with the app
